# **App Name**: Porobangla AI

## Core Features:

- AI Note Generator: Generate teacher-style structured notes with headings, subheadings, bullet points, definitions, examples, and exam tips in English, Bengali, or mixed Bangla-English. Includes LaTeX support for math equations.  AI tool incorporates 'Exam Focus', 'Common Mistakes', and 'Memory Tricks' sections.
- AI Mock Test Generator: Generate mock tests with MCQs, short answers, long answers, and numerical problems with difficulty levels. Generates output in question paper style with marks, question paper, answer key, and detailed solutions.
- Chat with Vidyasagar (AI Tutor): An AI tutor persona specializing in Maths, Physics, and Chemistry.  It never gives direct answers but explains reasoning and asks follow-up questions. It provides analogies from daily Indian life.
- AI Study Planner: Generate a day-wise timetable with revision slots and mock test days, adjustable based on user progress and missed days.
- Notebook-style UI: Center-focused layout with soft colors, rounded cards, smooth animations, and glassmorphism.
- Printable Notes: Notes formatted to be copy-friendly and PDF-ready.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF), evoking calmness and focus, appropriate for learning.
- Background color: Very light blue (#F0F8FF), which subtly sets off the primary color, while avoiding stark contrast.
- Accent color: Pale purple (#BDB2FF) for interactive elements and highlights, giving the app a modern feel.
- Headline font: 'Playfair', serif, for an elegant, fashionable feel, appropriate for the high level of educational content; body font: 'PT Sans', sans-serif, for readability
- Code font: 'Source Code Pro' for displaying LaTeX equations and code snippets.
- Center-focused layout, input on top, output below, mimicking a notebook style.
- Smooth, subtle animations for transitions and interactive feedback.
- Dark background with starlight effect.
- Large, impactful headline text.
- Minimalist design with clear call-to-action buttons.
